# mood_based_song_recommendation
It's a project where user gives their mood as an input either by text or using dropdown list the model recommends the songs with artist name and movie name as an output where the output is presented in the webpage using streamlit. 
I'm using the dataset here a telugu dataset so you can download it if you want.
This is the project where you can access this project if you like it once try it.
The project aim is to recommends the songs based on their moods because we are the people always loves listening music if it is any situation so, based on that situation mood when the user enter or select mood then they will get their mood based songs in the webpage.
